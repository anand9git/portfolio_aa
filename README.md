# Anand Aggarwal - Portfolio Website

Professional portfolio website showcasing experience as a Backend Engineer at Adobe.

## Features

- **Fully Responsive** - Works on mobile, tablet, and desktop
- **Professional Design** - Clean blue color scheme
- **Performance Optimized** - Fast loading and smooth animations
- **SEO Ready** - Proper meta tags and semantic HTML
- **Accessible** - WCAG compliant with keyboard navigation

## Quick Start

Simply open `index.html` in your browser, or use a local server:

```bash
python3 -m http.server 8000
# Visit: http://localhost:8000
```

## Color Customization

Edit CSS variables in `styles.css` (line 2):

```css

    --primary: #2563eb;      /* Main color */
    --accent: #06b6d4;        /* Accent color */
}
```
